# AI Development Workflow for Software Engineers

A collection of reusable AI prompts designed to support daily software engineering activities.

These prompts are technology-agnostic and can be used with:

- GitHub Copilot
- ChatGPT
- Claude
- Gemini
- Cursor
- Windsurf
- Any AI coding assistant

The goal is to help engineers:

- Understand requirements
- Implement features safely
- Fix bugs efficiently
- Review code effectively
- Identify risks earlier
- Improve self-validation before QA
- Reduce regressions
- Increase productivity

---

# Who Is This For?

- Software Engineers
- Frontend Developers
- Backend Developers
- Full Stack Developers
- Technical Leads
- Engineering Teams

# Prompt Library

| Prompt | Purpose |
|----------|----------|
| 01 Requirement Analysis | Understand requirements before development |
| 02 Feature Implementation | Implement new functionality |
| 03 Bug Fix | Analyze and fix defects |
| 04 Code Analysis | Understand existing code |
| 05 Code Review | Review implementation quality |
| 06 Test and Risk Analysis | Identify missing scenarios and risks before QA |
| 07 Self Review and Fix | Self-review and improve implementation |
